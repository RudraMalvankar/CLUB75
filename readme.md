# 📖 Overview

Club75 is a beautifully designed, offline-first attendance management application built for students.

Unlike traditional attendance trackers, Club75 helps students understand their attendance, predict future percentages, calculate safe bunks, and make informed academic decisions.

Whether you're wondering:

- Can I skip tomorrow?
- How many lectures do I need to attend?
- Will I stay above 75%?
- Which subject is putting me at risk?

Club75 answers these instantly.

---

# ✨ Vision

We believe attendance tracking should be effortless.

Club75 aims to become the operating system for college students by combining attendance management, intelligent insights, planning tools, and productivity features into one seamless experience.

Today it's attendance.

Tomorrow it's your academic companion.
