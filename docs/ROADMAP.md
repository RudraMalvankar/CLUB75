# 🗺 Club75 Product Roadmap

Version: 1.0

Status: Living Document

---

# Purpose

This roadmap defines the development journey of Club75.

Its purpose is to ensure that the product grows in a structured manner without feature creep while maintaining high quality.

Every new feature should be evaluated against this roadmap before implementation.

# MVP (Version 0.1)

Goal

Build a usable offline attendance application.

Features

- Beautiful onboarding
- Semester setup
- Subject management
- Timetable
- Attendance tracking
- Dashboard
- Dark Mode
- AMOLED Mode
- Settings

Success Criteria

A student can completely replace manual attendance tracking.
